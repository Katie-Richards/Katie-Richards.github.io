HTML

HTML is the structure of the webpage, below is some basic tags to get started with designing your first website.

Basic Tags

<html> </html> Creates a HTML document
<head> </head> Sets off title and other information that isn't displayed
<body> </body> Sets off the visible portion of the document
<title> </title> Puts name of the document into the title bar
<link> </link> Links any resources needed to the html file eg.stylesheet, font-family

Text Tags

<h1> </h1> to <h6> </h6> Creates headings eg.h1 largest h6 smallest
<em> </em> Emphasizes a word in italics
<strong> </strong> Emphasizes a word in bold

Links

<a href= "URL"> </a> Creates link to specified source

Formatting

<p> </p> Creates a new paragraph
<br> Inserts a line break note break line tag is self closing
<div> </div> Creates a block division
<span> </span> Creates an inline division

Lists

<ul> </ul> Creates an unordered list
<ol> </ol> Creates an ordered list
<li> </li> Surrounds each list item

Media

<img scr= "URL"> Inputs an image note imaage tag is self closing
<figure> </figure> Encompasses an image and its caption
<figcaption> </figcaption> Assigns a caption to an image
<video src= "URL" control>Video not available </video> Inputs video
<audio src="URL" control>Audio not available </audio> Inputs audio

CSS


CSS is what is used to style the website and make it look good. Below is some basic properties to get you started.


Selectors Properties and Values

Selectors are the elements on which the style should be applied to

Property is the style catergory in which you will be changing

Value is the specific style that will be applied


Attching your Stylesheet to your HTML file

To attach your stylesheet to your html file you will need to save your stylesheet within your workspace folder as style.css.

Then link your stylesheet toy your html file by using the following link in the head section of the file <link rel= "stylesheet" href= "./style.css">


Basic Seceltors

Element name The original name of the element eg. header, h1, p
.class name Class specified in the element eg. <h1 class= "heading">
#id name ID specified in the element eg. <h1 id= "main">
* Encompasses the enitred document
